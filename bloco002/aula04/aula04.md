# Aula 04 - Comentários


#### Comentário de uma linha
```js
// - comentário de uma linha
```

#### Comentário de várias linhas
```js
/*
    Comentário
        de 
        várias 
        linhas
*/
```

#### Comentário de várias linhas (documentação)
```js
/**
 * Comentário
 * de
 * várias
 * linhas
 * recomendado
 * para
 * documentação 
 */ 
```
